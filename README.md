# 个人博客

本仓库用于整理与发布个人技术文章，以 Markdown 形式存放，按日期分目录管理。

## 内容方向

文章主要围绕以下主题：

- **后端**：服务架构、性能、工程实践等
- **Kubernetes**：集群、调度、GPU / 设备插件与相关运维话题
- **LLM**：大模型相关的基础设施、推理与落地场景

具体篇目见 [`posts/`](posts/) 下的目录与文件。

## 仓库结构

```
posts/
  YYYYMMDD/          # 按发布日期归档
    *.md             # 文章正文
    *.png 等         # 文中引用的配图（如有）
```

## 使用说明

- 文章为普通 Markdown，可用任意编辑器阅读；若需本地预览渲染效果，可使用支持 CommonMark / GFM 的预览工具或静态站点生成器自行挂载该目录。
- 文内图片请使用相对路径引用，并与对应文章放在同一日期目录下，便于整体迁移与归档。
